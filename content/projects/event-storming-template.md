## 1. Elevator Pitch

A ready-to-use Event Storming template for Diagrams.net that speeds up domain discovery workshops and cross-functional collaboration.

## 2. Problem and Context

Discovery sessions often lose time on setup and visual consistency. This project provides a reusable library to start modeling immediately.

## 3. Scope and Role

- Visual library design.
- Direct integration publishing for Diagrams.net.
- Open community maintenance and contribution support.

## 4. Solution Implemented

A custom Draw.io/Diagrams.net library distributed by URL, built around Event Storming modeling practices.

## 5. Stack and Architecture

- XML template for Diagrams.net.
- Public distribution through GitHub.
- MIT license.

## 6. Quality and Engineering Practices

- Clear usage documentation.
- Reuse-first design.
- Community adoption signal (stars/forks).

## 7. Portfolio Relevance

Shows practical maturity in DDD/Event Storming and ability to create reusable assets that improve team discovery workflows.
