A full-stack planner for electronic fuel injection wiring harnesses.

Highlights:

- Backend in NestJS with Prisma and JWT authentication.
- Frontend in React + Vite + TypeScript.
- Domain features for project planning and build recipe generation.
- Subscription-tier logic with feature/limit enforcement.
