MapEFI is a full-stack project that combines a structured Java backend with a modern Next.js frontend.

Highlights:

- Monorepo-style organization with clear backend/frontend boundaries.
- Backend organized for maintainability with layered modules.
- Frontend with TypeScript tooling, Storybook, and testing setup.
- Documentation and architecture decision records to support team execution.
