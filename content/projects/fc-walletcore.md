WalletCore is an event-driven architecture project centered on transaction and balance processing.

Highlights:

- Kafka-based asynchronous communication between services.
- Mixed-service ecosystem with Go and Java components.
- Docker Compose environment for local orchestration.
- Practical demonstration of EDA concepts and integration flows.
